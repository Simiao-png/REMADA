# Grade Horária

Sistema web para geração automática de grades horárias escolares.

## Objetivo

Automatizar a construção de grades horárias respeitando disponibilidade de professores, carga horária das disciplinas e restrições da escola.

## Tecnologias

- Python
- Flask
- PostgreSQL
- Git/GitHub
